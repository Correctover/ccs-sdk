# Contributing to ccs-sdk

## 重要声明 — 闭源商业产品

**ccs-sdk 是闭源商业付费产品，不是开源项目。**

本仓库仅作为展示窗口，源代码不公开。

## 不接受外部 Pull Request

本仓库不是社区项目，不接受外部 PR。

## 反馈渠道
- 安全漏洞报告: security@correctover.com
- Bug: GitHub Issues
- 商业合作: wangguigui@correctover.com

## 内部开发规范
- Python 3.10+, 类型注解
- 遵循 pyproject.toml lint 规则
- 必须有单元测试
